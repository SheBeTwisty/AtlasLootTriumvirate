## AtlasLootTriumvirate

!! This addon is currently in Alpha and errors may occur !!


# What is this?

This is a modified version of AtlasLoot v5.11.04 by Hegarol for the Triumvirate-wow server.


# How do I install this?

1. Remove any existing AtlasLoot folders from your /Interface/AddOns folder
2. Download the repository and unzip it
3. Copy the AtlasLoot folders from the master-folder into /Interface/AddOns
